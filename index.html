BASIC HTML

Html is a language used to make words on your website. Html stands for Hypertext Markup Language. Html uses tags to control where to put the writing. A tag can make the text bolder or make a title. Tags are used in many ways. A basic html document always starts with <html> and ends with </html>. The slash is there to indicate that it's the closing tag so the program won't confuse it with something else.

Tag List:

  • <html> starting a document
  • <head> this will make the text show at the top of the page
  • <title> this is to change the tab title
  • <h1> this will make the text into a heading(the number changes the size)
  • <p> this separates each paragraph
  • <body> this will show where the body of a text is
  • <i> this makes text become italic
  • <a href="Insert Url"> with this you can add links
  • <u> this tag makes the text underline
  • <div> it divides the content
  • <b> makes the text bold
